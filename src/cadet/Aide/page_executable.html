Page Exécutable

Ce menu permet l'affichage du code machine qui est générer aprés la compilation de l'expression arithmétique(fichier codeDASM.txt)

  1. Le logiciel reçoit en entrée une expression arithmétique parenthésée construite correctement.
  2. Le logiciel traite cette expression arithmétique en la traduisant en une nouvelle expression arithmétique postfixée.
  3. Le logiciel compile l'expression arithmétique postfixée en la traduisant en une liste d'instructions de machine à pile MAP et en un fichier contenant le codage de chaque instruction de la liste. Ce dernier fichier représente un "exécutable" que le logiciel peut exécuter indépendamment. Le logiciel à cette étape de son travail engendre deux fichiers : le fichier "codegenere.txt" et le fichier "codegenere.exec"

Le fichier "codegenere.txt" contient les résultats en mode texte de la traduction postfixée

Exemple du fichier codeDASM.txt
