Page d'accueil

Présentation de l'interface utilisateur

La page d'accueil de l'application se présente avec un choix d'utilisation:

  • Des menus déroulants avec les raccourcis clavier :

    • Un menu Fichier pour l'ouverture et/ou l'enregistrement d'un fichier compiler
    • Un menu éditionEdition agissant sur les zones texte
    • Un menu Traitements pour les changement d'écrans
    • un menu Affichage permettant une personnalisation de l'interface, notamment pour le choix de langue
    • Un dernier menu d'Aide

  • Une série de boutons à gauche, pour une utilisation directe de l'application.

liens panneau gauche

Boutons

  1. Présentation :Le bouton présentation permet de revenir à la page d'accueil
  2. Saisie : Donne accès à l'écran de saisie des expressions arithmétiques
  3. Rapport : Affiche le code assembleur
  4. Executable : Affiche le code machine
  5. Fichier XML : affiche le fichier XML
  6. Désassemble : Affiche les deux fichiers codeDASM.txt et codegenere.txt
  7. Animation : Permet de visualiser l'animation du programme compiler
  8. Documentation : Donne accès à toute l'aide de l'application
  9. Et enfin un bouton A Propos
